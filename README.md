## **🫀 How to play?**
|Key|Explanation|
|:---:|:---:|
|**← , → , ↓**|Move block.|
|**↑**|Rotate the block clockwise.|
|**C**|Hold block or change the current block to hold block.|
|**Space bar**|Hard drop (place block immediately)
